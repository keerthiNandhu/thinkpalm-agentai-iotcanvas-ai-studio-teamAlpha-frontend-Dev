# IoTCanvas AI Studio

AI-powered IoT dashboard generator using a multi-agent pipeline.

## Structure

```
src/
├── app/               # Next.js App Router
├── components/        # UI components
├── agents/            # AI agent classes
├── memory/            # Memory service
├── tools/             # Agent tools
├── widgets/           # IoT widget components
├── hooks/             # React hooks
├── services/          # Business services
├── types/             # TypeScript types
└── lib/               # Shared utilities
```

## Getting Started

```bash
npm install
npm run dev
```
